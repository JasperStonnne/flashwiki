  ### 数据流

  1. 组件挂载 → 并行请求:
     - `GET /api/me` → 获取当前用户 ID（用于判断"不可操作自己"）
     - `GET /api/admin/users` → 获取用户列表
  2. 两个请求都完成后渲染表格
  3. Loading 态: 居中 "加载中..." 文字
  4. Error 态: 红色错误提示 + "重试" 按钮
  5. 空态: 居中 "暂无用户"

  ### 表格列

  | 列 | 内容 | 样式说明 |
  |---|---|---|
  | 头像 | 渐变色圆角方块 + display_name 首字母 | 32px, border-radius var(--radius-sm)，背景用 userId hash 选色 |
  | 邮箱 | email | font: 400 14px var(--font-cn)，color var(--gray-700) |
  | 显示名 | display_name | font: 500 14px var(--font-cn)，color var(--gray-800) |
  | 角色 | pill badge | Manager: background var(--primary-bg)，color var(--primary); Member: background var(--gray-100)，color var(--gray-500); font: 500 12px var(--font-ui)，padding 2px 10px，border-radius var(--radius-full) |
  | 注册时间 | created_at 格式化为 YYYY-MM-DD | font: 400 13px var(--font-ui)，color var(--gray-400) |
  | 操作 | 按钮组 | 见下方操作列 |

  ### 操作列

  每行最多两个操作按钮。当该行是当前登录用户时，两个按钮都不渲染（整列空白）。

  1. **改角色**
     - 当前是 Member → 按钮文字 "册封 Manager"
       - 点击 → PATCH /api/admin/users/:id/role body { role: "manager" }
     - 当前是 Manager → 按钮文字 "撤销 Manager"
       - 点击 → PATCH /api/admin/users/:id/role body { role: "member" }
     - 样式: Secondary 按钮风格（白底、gray-200 边框、gray-600 文字）
     - 请求中: 按钮 disabled + 文字变 "处理中..."
     - 成功 → 就地更新该行 users state 中的 role（不可变更新，不重新拉列表）
     - 失败 → 设置该行的 rowError state，显示红色行级错误文字

  2. **强制下线**
     - 按钮文字 "强制下线"
     - 样式: Danger 按钮（design-system §7.1）:
       - 默认: background transparent，border 1.5px solid var(--rose)，color var(--rose)
       - hover: background var(--rose)，color var(--white)
     - 点击 → POST /api/admin/users/:id/force-logout（无 body）
     - 请求中: disabled + 文字变 "处理中..."
     - 成功 → 行级提示 "已强制下线"（绿色文字，3 秒后自动消失）
     - 失败 → 行级红色错误文字

  ### 行级反馈

  - 每行维护独立的 feedback state: { type: 'success' | 'error', message: string } | null
  - 成功反馈 3 秒后自动清除（setTimeout）
  - 错误反馈手动清除（下次操作时替换）
  - 渲染位置: 操作列下方，font: 400 12px var(--font-cn)
  - 成功: color var(--green)
  - 错误: color var(--rose)

  ### 文件 3: `frontend/src/pages/AdminUsersPage.tsx`（改写）

  当前仅占位 `<h1>Admin Users</h1>`，需改为完整页面。

  ```
  Props: 无（页面组件）

  内部状态:
  - users: UserResponse[]           — 用户列表
  - currentUserId: string | null    — 当前登录用户 ID（来自 GET /api/me）
  - loading: boolean                — 初始加载中
  - loadError: string | null        — 加载错误信息
  - rowFeedback: Record<string, { type: 'success' | 'error', message: string }>
                                    — 行级反馈（key 为 user.id）
  - busyRows: Set<string>           — 正在请求中的行 ID（用于 disable 按钮）

  初始化（useEffect，依赖 []）:
  - 并行发起两个请求:
    const [meData, usersData] = await Promise.all([
      request<UserResponse>('GET', API_ENDPOINTS.me),
      request<UserResponse[]>('GET', API_ENDPOINTS.adminUsers),
    ])
  - 成功: setCurrentUserId(meData.id)，setUsers(usersData)，setLoading(false)
  - 失败: setLoadError(error.message)，setLoading(false)

  handleChangeRole(userId: string, newRole: UserRole):
  - setBusyRows(prev => new Set(prev).add(userId))
  - clearRowFeedback(userId)
  - 调用 request<null>('PATCH', API_ENDPOINTS.adminUserRole(userId), { role: newRole } satisfies ChangeRoleRequest)
  - 成功:
    - setUsers(prev => prev.map(u => u.id === userId ? { ...u, role: newRole } : u))
    - setRowFeedback({ ...rowFeedback, [userId]: { type: 'success', message: '角色已更新' } })
    - setTimeout 3s 后清除该行 feedback
  - 失败:
    - setRowFeedback({ ...rowFeedback, [userId]: { type: 'error', message: error.message } })
  - finally: setBusyRows(prev => { const next = new Set(prev); next.delete(userId); return next })

  handleForceLogout(userId: string):
  - setBusyRows(prev => new Set(prev).add(userId))
  - clearRowFeedback(userId)
  - 调用 request<null>('POST', API_ENDPOINTS.adminUserForceLogout(userId))
  - 成功:
    - setRowFeedback({ ...rowFeedback, [userId]: { type: 'success', message: '已强制下线' } })
    - setTimeout 3s 后清除该行 feedback
  - 失败:
    - setRowFeedback({ ...rowFeedback, [userId]: { type: 'error', message: error.message } })
  - finally: setBusyRows(prev => { const next = new Set(prev); next.delete(userId); return next })

  渲染结构:
  <div className="admin-users">
    <h1 className="admin-page-title">用户管理</h1>

    {loading && <div className="admin-empty">加载中...</div>}

    {loadError && (
      <div className="admin-error">
        <p>{loadError}</p>
        <button className="btn-secondary" onClick={retry}>重试</button>
      </div>
    )}

    {!loading && !loadError && users.length === 0 && (
      <div className="admin-empty">暂无用户</div>
    )}

    {!loading && !loadError && users.length > 0 && (
      <div className="users-table-wrapper">
        <table className="users-table">
          <thead>
            <tr>
              <th></th>           {/* 头像列，无标题 */}
              <th>邮箱</th>
              <th>显示名</th>
              <th>角色</th>
              <th>注册时间</th>
              <th>操作</th>
            </tr>
          </thead>
          <tbody>
            {users.map(user => (
              <tr key={user.id} className="users-row">
                <td><UserAvatar user={user} /></td>
                <td className="users-email">{user.email}</td>
                <td className="users-name">{user.display_name}</td>
                <td><RoleBadge role={user.role} /></td>
                <td className="users-date">{formatDate(user.created_at)}</td>
                <td className="users-actions">
                  {user.id !== currentUserId && (
                    <>
                      <button
                        className="btn-secondary btn-sm"
                        disabled={busyRows.has(user.id)}
                        onClick={() => handleChangeRole(
                          user.id,
                          user.role === 'manager' ? 'member' : 'manager'
                        )}
                      >
                        {busyRows.has(user.id) ? '处理中...'
                          : user.role === 'manager' ? '撤销 Manager' : '册封 Manager'}
                      </button>
                      <button
                        className="btn-danger btn-sm"
                        disabled={busyRows.has(user.id)}
                        onClick={() => handleForceLogout(user.id)}
                      >
                        {busyRows.has(user.id) ? '处理中...' : '强制下线'}
                      </button>
                    </>
                  )}
                  {rowFeedback[user.id] && (
                    <span className={
                      rowFeedback[user.id].type === 'success'
                        ? 'row-feedback row-feedback--success'
                        : 'row-feedback row-feedback--error'
                    }>
                      {rowFeedback[user.id].message}
                    </span>
                  )}
                </td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>
    )}
  </div>
  ```
