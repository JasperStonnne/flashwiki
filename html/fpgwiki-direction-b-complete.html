一、项目背景

FPGWiki 旨在为中小团队提供轻量、高效的在线文档协作平台。当前市场上的解决方案普遍存在功能冗余、权限粒度粗的问题，我们的差异化定位是提供「刚好够用」的协同体验。

核心价值主张：让团队知识流动起来，而不是沉睡在本地文件夹中。

二、核心功能

  • 实时协同编辑：基于 CRDT 算法，支持多人同时编辑同一文档
  • 四级权限体系：manage → edit → readable → none
  • 版本历史：自动保存版本快照，支持一键回溯
  • 全文搜索：基于 Elasticsearch，支持中文分词

三、技术选型

前端采用 React 18 + TipTap 编辑器框架，后端使用 Go + PostgreSQL，协同层通过 WebSocket + Y.js 实现。
